# Challenge System - Page Design Specifications

## Design System (Keep Consistent)

### Colors
- **Background**: `bg-dark-bg-soft` (#0A0A0A)
- **Text**: White (#FFFFFF)
- **Accents**: `bg-brand-yellow` (#FBBF24) for CTAs
- **Cards**: `bg-warm-stone-secondary` (#E7E3DD)
- **Dramatic**: `bg-charcoal-blue-dark` (#1A2533)
- **Borders**: `border-warm-stone-border` (#D6D2CB)

### Typography
- **Headings**: Font black, tracking-tight
- **Subheadings**: Font bold, text-gray-300
- **Body**: Font light, text-gray-300

### Components
- Animations: Framer Motion
- Rounded corners: lg to xl
- Shadows: shadow-lg to shadow-2xl
- Transitions: duration-300 to duration-500

## Design Implementation Guidelines

### Consistency Rules
1. **Header**: Always use Navbar with fixed positioning
2. **Footer**: Always include Footer component
3. **Background**: Always use `bg-dark-bg-soft`
4. **Text**: White for primary, gray-300 for secondary
5. **Cards**: Use `bg-warm-stone-secondary` for content cards
6. **Buttons**: Yellow (`bg-brand-yellow`) for primary CTAs
7. **Animations**: Use Framer Motion for all transitions
8. **Spacing**: Consistent padding (px-6 lg:px-12)
9. **Shadows**: shadow-lg for cards, shadow-2xl for featured

### Responsive Design
- Mobile: Single column (1 col)
- Tablet: Two columns (lg:grid-cols-2)
- Desktop: Three columns (lg:grid-cols-3)
- Large screens: Four columns (lg:grid-cols-4) where applicable

### Typography
- Hero titles: text-5xl lg:text-6xl font-black
- Section titles: text-3xl lg:text-4xl font-black
- Card titles: text-xl font-black
- Body text: text-sm md:text-base font-light

### Color Usage
- Backgrounds: dark-bg-soft, charcoal-blue-dark
- Cards: warm-stone-secondary, white
- Text: white, black, gray-300, gray-600
- Accents: brand-yellow
- Borders: warm-stone-border

### Animation Patterns
- Hero sections: fade-in + scale
- Cards: slide-in + hover scale
- Buttons: color change + scale on hover
- Lists: stagger animation for items
- Modals: fade-in + slide-from-top
